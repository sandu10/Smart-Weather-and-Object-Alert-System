# Smart Weather & Object Alert System

Arduino project using DHT22 and Ultrasonic sensors to monitor temperature, humidity, and object distance.  
Provides visual (LEDs) and audible (buzzer) alerts and displays info on a 16x2 LCD.

## Components
- Arduino Uno
- DHT22 Temperature/Humidity sensor
- HC-SR04 Ultrasonic sensor
- 16x2 LCD (I2C)
- Red and Green LEDs
- Buzzer

## Features
- Displays temperature, humidity, and distance in real-time

- Alerts for nearby objects (<10 cm)
